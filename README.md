# Github-Repository-Getter
An Terminal Based Application that can download repositories from someone or see all their repositories with the info

## How To Use

<sapn> 1. First Of All in your terminal switch to the folder that this program is downloaded in including the file `__init__.py` </span> <br>
<span> 2. You can type `python __init__.py -download "user" "repository"` to clone or download a repository </span> <br>
<sapn> 3. You can type `python __init__.py -getInfo "user"` to to get information about a all repositories for a user </span> <br>

## Good to note
>[!WARNING]
>This Is the first version so the application is not complete, it doesn't have strong error handelings and it also can not download a repository with one or multiple folders ( It can download the other files but not the folders )
